- Java
- Spring Boot

## Deployment in Production Environment
- Back end: Heroku
- Database: Postgresql

# How to run the project

## Back end
Prerequisites: Java 11

```bash
# clone repository
git clone https://github.com/developer-rrm14/rrm14-ms-matriz-curricular.git

# run the project
./mvnw spring-boot:run
```
